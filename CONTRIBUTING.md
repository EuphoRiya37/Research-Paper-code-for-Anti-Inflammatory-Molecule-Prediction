# Contributing to Research Paper Code

Thank you for interest in this published research!

## Setup for Researchers

```bash
git clone [repo]
cd research-molecule-prediction
pip install -r requirements.txt

# Train model
python train.py --config configs/gat_default.yaml
```

## What to Work On

- Benchmark against new datasets
- Implement alternative architectures
- Improve interpretability
- Extend to other molecular properties
- Optimize for deployment
- Add new evaluation metrics

## Before Submitting

- Test reproducibility
- Document methodology
- Compare against baselines
- Add comments for clarity
- Update requirements if needed

## Reporting Issues

- Describe the problem
- Include error messages
- Share reproducible example
- Mention Python & package versions

## Questions?

Open an issue to discuss!
